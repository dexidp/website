{{- /* Render hook for markdown headings (h1-h6). Emits: {text} The trailing is a real element, not a pseudo, so its click target is isolated from the parent heading's hit box and never steals clicks from adjacent paragraph links. h1 is emitted without an anchor (it's the page title — nobody links to it). Headings that already contain a link inside (rare but possible in markdown like `## [Title](url)`) also skip the anchor to avoid double links. */ -}} {{- $hasLink := or (findRE "" .Text) -}} {{- .Text | safeHTML -}} {{- if and (ge .Level 2) (not $hasLink) -}} {{- end -}}
